This project is based on Selenium, TestNG, Mavens with Java as the language

Execute TestNG file to execute the project.

src-main-java-pageObjectClasses contains all the pageObject classes
src-test-java contains the test classes

src-main-java-utility contains the utility Property classes

config.properties contains the url of the project. As of now it contains only one parameter (url), but this was just to showcase the usage

testng.xml - use this file to run the program. It contains the test methods to run and the classes to include.

pom.xml - this contains all the dependencies that this project needs
